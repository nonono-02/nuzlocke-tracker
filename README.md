# NuzTrack - Pokemon Nuzlocke Tracker

NuzTrack is a guide for players to use when playing Nuzlockes of Generation 3 games (Ruby, Sapphire, Emerald, Fire Red, and Leaf Green). 

All major boss fights are included, as well as rival and league battles. Future updates will add all battles within the game (optional or not).

All assets and logos are owned by Nintendo and The Pokemon Company. Sprites are provided by PokeAPI.
